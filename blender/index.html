My Blender Projects

This is a documentation of all the projects I have done in blender and I hope to continue to experiment more and create more wonderful things and animations!

This was my ritual object, a lantern that symbolizes good luck when hung up during Chinese New Year. I used the extrude tools to create the elements of the lantern and used the shading tools to make certain elements more metallic/gold-like.

This was my quotidian object, a mug I use daily. I created it with curving tools on Blender — something I've never done before.

This was my version of an Animated Joseph Cornell Box and I decided to incorporate some of the different objects I had saved. This was my first time applying physics to the objects so they didn't just fall through the box.

This was my first time trying to create a sunrise over a scene I made and it took me a while to figure out the right objects to parent together to create the sense of a sunrise to sunset scene.

This was my first project in Blender and I created the object through writing out a vertexlist to create a play button.
